# Study-Ground
Study-Ground는 매일 공부한 내용을 실습해보는 공간이다. 각 폴더는 공부 항목이며, `specification`과 `code` 부분으로 나누어진다. 
+ `Specifications` : 그 날 작성한 코드의 목적을 명확하게 정리하는 문서이다. 
+ `Code` : 작성 코드이다. 
코드 작성이 끝난 뒤에는 실행 결과와 specification 사이에 발생한 차이의 이유를 정리한다.
